# Spelling Rules

## British-Specific Spellings

- **Colour**: Use 'colour' instead of 'color'.
- **Favour**: Use 'favour' instead of 'favor'.
- **Centre**: Use 'centre' instead of 'center'.
- **Organise**: Use 'organise' instead of 'organize'.
- **Realise**: Use 'realise' instead of 'realize'.
- **Defence**: Use 'defence' instead of 'defense'.
- **Licence**: Use 'licence' instead of 'license' (noun).
- **Cheque**: Use 'cheque' instead of 'check' (financial instrument).
- **Grey**: Use 'grey' instead of 'gray'.
- **Programme**: Use 'programme' instead of 'program' (except when referring to computer programs).
