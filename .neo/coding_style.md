# Coding Style

## C# Naming Conventions

- **Classes and Enums**: Use PascalCase.
  - Example: `CustomerOrder`, `OrderStatus`

- **Methods**: Use PascalCase.
  - Example: `CalculateTotal()`, `ProcessOrder()`

- **Local Variables and Parameters**: Use camelCase.
  - Example: `orderTotal`, `customerName`

- **Constants**: Use PascalCase.
  - Example: `MaxItems`, `DefaultTimeout`

- **Interfaces**: Prefix with 'I' and use PascalCase.
  - Example: `IOrderProcessor`, `ICustomerRepository`

- **Private Fields**: Use camelCase, optionally prefixed with an underscore (_).
  - Example: `_orderRepository`, `customerList`
